# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-03

### Added
- Initial release: dual-mode PHP Debugger Forensics extension
- DevTools panel for network forensics and failure analysis
- Popup UI for user-friendly bug recording and export
- Support for correlation ID detection and lookup
- Incident bundle export (JSON + sanitized metadata)
- Full Nuxt 3 + TypeScript + Tailwind scaffold
- Chrome Manifest V3 configuration
- GitHub Actions workflow for automated releases
- Comprehensive README and documentation
- MIT License

### Known Limitations
- Manifest V3 limitations on response body inspection
- Firefox support planned
- Service worker must use message passing
